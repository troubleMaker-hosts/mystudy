# mystudy
学习java时的一些demo

jdk1.8(1.6需要修改部分代码(加密算法中)),jdk11兼容会隔断时间做一次(不固定).

本项目主要是本人在工作中遇到一些问题时,会在此项目中先写一个小demo,便于工作中遇到此类问题能较快上手.
本项目study包中主要是一些数据结构,算法之类的一些demo.比较基础,时本人初学软件编程时写的.
